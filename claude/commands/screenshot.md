# /screenshot — Take and analyze a browser screenshot

Capture the current browser state and analyze what's visible.

## Steps

1. Call `take_screenshot` to capture the current page.
2. Read the screenshot image using the Read tool.
3. Describe what you see: page layout, visible content, any visual issues (broken layout, missing elements, loading spinners stuck, error messages).
4. If the user provided a specific concern (e.g., "the button is misaligned"), address it directly.
